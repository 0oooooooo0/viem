# removeBlockTimestampInterval

Removes `setBlockTimestampInterval` if it exists.

## Import 

```ts
import { removeBlockTimestampInterval } from 'viem'
```

## Usage

```ts
import { removeBlockTimestampInterval } from 'viem'
import { testClient } from '.'
 
await removeBlockTimestampInterval(testClient) // [!code focus]
```
